Volo Bible : Settings

Created by Jocelyne Kizziar, last modified on Jun 15, 2020

The Setting Section is where the hotel location and appearance are set or changed along with other important details. 

  1. Click Settings in the main Content Menu

  2. Enter setting data into each of the following fields (see fig 4).

    1. Color:  Using a hexadecimal value, enter the main application color scheme agreed upon or that has been lifted from the brand’s website.  This value colors the main menu buttons on the front page and the writing and icons displayed within the front-end interface (see fig 5).

    2. Logo:  This logo field will allow the hotel logo to be displayed on the main screen of the tablet.  As a general rule, the logo that is uploaded should be a white transparent file. The file must be a .png and should be greater than 600x300 pixels.

      1. Choose or Upload the main logo using the process described in this manual.

      2. Ensure that the file is .png format and greater than 600 x 300 pixels

    3. Small Logo:  The small logo is displayed on all pages and sections and serves as the “home” navigation button.

      1. Choose or Upload the main logo using the process described in this manual.

      2. Ensure that the file is .png format and greater than 200 pixels in height.

    4. Background: This is the image displayed on the main screen of the tablet.

      1. Choose or Upload the main image using the process described in this manual.

      2. Ensure that the file is .jpg format and 1024 x 768 pixels

    5. Check “Enable products ordering and messages” for hotels operating in DHD+Messaging and Full Interactive modes.  Leave this box unchecked if hotel is operating in DHD mode.

    6. Checkout time:  The tablets are designed to clear the cache every day at the hotel checkout time.  This erases guest messages located in the inbox.

    7. Service recovery section:  For DHD + Messaging and Full Interactive, the tablet is scheduled to ask the hotel guests how they are enjoying their stay each day.  The service recovery denotes which section will receive notifications for “thumbs down” votes.

    8. Google Analytics 


  3. Enter Location data for the hotel.  The location of a hotel is needed for the weather display on the main screen of the front-end interface.  The weather is updated every fifteen minutes (see fig 6).

    1. Use the Search Hotel Location box to enter the hotel name or hotel address.  Being specific as possible ensures the weather will be correct at the location.

    2. Timezone:  Choose the hotel’s timezone from the drop-down box.

    3. Choose from Celsius or Fahrenheit degree format from the drop-down box. 









  4. Enter Notifications data. Hotels operating in DHD + or Full Interactive mode can have order messages go to an email address and/or as a text message to a phone number in addition to showing up in the staff-end interface.

    1. Phone: Add phone number of contact

    2. Email: Add the email address of the contact

  5. Check Advanced Settings are set.

    1. Check Enable cache

    2. Uncheck Set maintenance mode in all devices

    3. Set Notify unprocessed orders time. The system will send a message to a guest if an order has not been processed in an allotted amount of time. This can be disabled by typing 0.

  6. Integrations: Synergy - Enter integration id

  7. Integrations: Impala - Enter integration id

  8. Integrations: Alice - Enter integration id
