Volo Bible : Adding Navigations

Created by Jocelyne Kizziar, last modified on Mar 23, 2020

Navigations create a pathway for content to be displayed on the front-end interface.  Each navigation created will be represented with either an icon or an image.

  1. Click Navigations +Add in the CAM

  2. Type the name of the navigation in the Title box that corresponds with the section, page, or product that has previously been created.

  3. Choose an icon from the drop-down menu for “generic sections” (e.g. housekeeping, maintenance, food sections, etc.)

  4. Choose (or upload) a picture for more specific navigations using the method described in this manual (e.g a restaurant, attraction, museum.).

  5. Choose “type of navigation” from the drop-down menu.  A navigation will be a page, section, product, store, or category of store and will correspond to the content you have previously created.

  6. Choose the “navigation” id from the drop-down menu.  This title will correspond to the Page, Product, Section, Store, or Category of Store that you have previously created.

  7. Save

    1. Click Save to exit the page

    2. Click Save and Continue editing to make changes to the existing page

    3. Click Save and add another to add a new navigation

Next Steps
