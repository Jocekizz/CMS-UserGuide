Volo Bible : Adding Pages

Created by Jocelyne Kizziar, last modified on Mar 23, 2020

Static Pages

Static Pages will be informational only and include a container for an image at the top half of the screen and space for descriptive text at the bottom half.

  1. Click Pages +Add in the CAM

  2. Title the Page according to the blueprints

  3. Leave the Product drop-down menu blank

  4. Button label:

  5. Content:

    1. Enter up to 1,175 characters or roughly 250 words of descriptive text or blurb in the Content section.  

    2. Add format to this content by using certain HTML code.  See the appendix for acceptable code to be used.

  6. Leave iframe blank.

  7. Choose or upload an image according to the process described in this manual.  (See Adding images) for image dimensions for pages.

  8. Feature labels: 

  9. Save

    1. Click Save to exit the page

    2. Click Save and Continue editing to make changes to the existing page

    3. Click Save and add another to add a new page

Next Steps

  1. Create a Navigation for this page

  2. Add Navigation to the appropriate Section using the method described in this manual.

Pages that display PDF or Web Page

The CMS also facilitates pages that display a web page or PDF. These pages must have a previously created product as a base this page. The following directions apply once a product has been created.

  1. Click Pages +Add in the CAM

  2. Choose the same title you used for the product associated with this page.

  3. Choose the product name from the drop-down menu to tie the product and the page together.

  4. Leave Button label blank.

  5. Leave Content blank.

  6. Select a PDF from the file browser, or enter the full URL of any external website to load into the iframe.  

  7. Disregard image entry.

  8. Save.

    1. Click Save to exit the page

    2. Click Save and Continue editing to make changes to the existing page

    3. Click Save and add another to add a new page

Next Steps

  1. Create a Navigation with the same name as the Product and Page.

  2. Add the Navigation to the appropriate section using the method described in this manual.
