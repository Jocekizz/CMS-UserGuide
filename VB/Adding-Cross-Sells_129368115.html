Volo Bible : Adding Cross Sells

Created by Jocelyne Kizziar, last modified on Mar 23, 2020

Cross Sells display as a sidebar on the Thankyou page to direct a guest to a different section or another service in an effort to increase the profitability and the usability of the tablets. Cross Sells should always navigate to a product, request, or service that has a cost if possible unless otherwise indicated in the blueprint.

  1. Select Cross Sells +Add in the CAM.

  2. Add a title that, ideally, is something catchy to grab guest attention.

  3. Add a description (action message) of product/service/request of about 100 characters.

  4. Choose or upload a Cross Sell image using the method described in this manual.

  5. Create a short caption (another action message).

  6. Create the Call to Action.

    1. Create a button label that will display on the navigation button.

    2. Choose where the button will navigate to by choosing the Section > Page from the navigation drop-down menu.

  7. Save.

    1. Click Save to exit the page.

    2. Click Save and Continue editing to make changes to the existing page.

    3. Click Save and add another to add a new Cross-Sell.
