Volo Bible : Adding Thankyou's

Created by Jocelyne Kizziar on Mar 23, 2020

Thankyou’s are the confirmation screens that display after a guest has ordered a product, service, or made a request.  Thankyou’s are added via a drop-down box when creating/editing a Product (Product specific) or when creating/editing a Section. This option will cover every request, product ,or service in that Section.

Entries in bold cannot be left blank!

  1. Select Thankyous +Add in the CAM.

  2. Create a title that will display prominently when a guest has made an order/request.

    1. Choose a title that corresponds to the service or request

    2. Choose “{{product.title}} requested” or “{{product.title}} service booked” for a general thankyou.  

  3. Create a Kicker that will display at the top of the thankyou page advising the guest the order/request went through. 

  4. Select “Order Thanks” from the Icon drop-down menu.

  5. Add a description that will convey the action and time frame for answering the request.

    1. Type simple text (e.g. “A technician will be with you shortly”)

    2. Add simple HTML code such as <b> bold</b>. See the Appendix for a list of acceptable HTML code.

    3. Add {{order.total_price}} to display the price that will be due on arrival or added to the guest folio.

  6. Create a Cancel Title - “Need to change or cancel….” is a good default.

  7. Add Cancel description notifying a guest what action to take if they need to make changes or cancel the order that was just made. 

  8. Save.

    1. Click Save to exit the page.

    2. Click Save and Continue editing to make changes to the existing page.

    3. Click Save and add another to add a new Thankyou.
