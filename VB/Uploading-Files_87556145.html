Volo Bible : Uploading Files

Created by Jocelyne Kizziar, last modified on Mar 09, 2020

Images are used in almost every aspect of the tablet to enhance the design and maximize usability. Content can consist of photographs, logos, videos, pdf images or iframes. Content accepted by the CMS are listed in the file browser and are as follows: 

 

  • Image (.jpg, .jpeg, .gif, .png, .tif, .tiff)

  • Document (.pdf, .doc, .rtf, .txt, .xls, .csv)

  • Video (.mp4, .mov, .wmv, .mpeg, .mpg, .avi, .rm)

  • Icon (.svg)

The max file size of an image accepted by the CMS is 50.0 MB. 

Content Dimensions

Images are sized in Photoshop for their specific type/location in order to display properly in the front-end interface.  Videos can be resized and resampled in a program such as Camtasia. The dimensions for images are as follows:


  • Logo: at least 600px wide

  • Small Logo: no more than 200px tall

  • Main Background Photo: 1024 px x 768 px

  • Page: 846 px x 320 px

  • Product: 239 px x 200 px

  • Product for Item Request/In-Room Dining: 92 x 95px

  • Cover Main (Video Poster) 726 px x 319 px

  • Cover Slider 298 px x 319 px

  • Navigation 318 px x 175 px

  • Staff - 174 px x 174 px

  • Cross Sells 193 x 193

  • Video sizing: 1282 x 564

  • iframe: 834px x 580px

To Upload Images

When prompted to upload an image/file when creating a page, navigation, product, cover:

  1. Click :search: to open the file browser. 

  2. Search for the image file using the search bar at the top of the FileBrowser if the file has been uploaded.  

  3. Upload an image by clicking the Upload button at the top right of the FileBrowser and choose one of two actions:

    1. Select files from hard drive by clicking “Upload a File” (You may upload more than one image at a time)

      1. Search computer hard drive for image file

      2. Select file and click open

    2. Drag and Drop files into the gray box marked “drop files here to upload”

  4. Wait to see that the image is done uploading by ensuring the red X is replaced with the black checkmark

  5. Click “Return to uploads” to return to the main FileBrowser Page.

  6. Click Select next to the file you want to add the CMS.  The FileBrowser will close once the image file has been selected.
